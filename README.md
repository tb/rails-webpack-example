# Rails5 Webpack

Example [webpack-rails](https://github.com/mipearson/webpack-rails) project

## Installing

    bundle
    cd frontend
    npm i

[Upgrading frontend packages](https://www.npmjs.com/package/npm-check-updates)

## Development

    rails s
    cd frontend
    npm start

## Production 

    RAILS_ENV=production rails s
    npm run build:prod
